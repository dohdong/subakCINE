# 프로젝트 소개
사용자가 영화 정보를 검색하고 평가할 수 있는 영화 정보 제공 웹사이트




# 프로젝트 주요기능
사용자가 원하는 영화나 TV를 선택하여 컬렉션을 생성할 수 있도록 제작


TMDB API를 활용하여 데이터를 가져와서 주요 페이지에 출력


HTML, CSS, JavaScript, JSP/서블릿, Ajax, Jquery, Git 활용
